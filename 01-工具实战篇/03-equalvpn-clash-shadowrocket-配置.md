# 一、购买可直连纯净住宅IP

点击 [EqualVPN](https://www.equaldcdn.com/)，后点击 Get started，按照指引先注册账号，然后再买套餐就行。

我买的是 50G/月 38.4￥ 的套餐，支付可以使用支付宝，非常友好！

购买完成后点击「开始使用」->「开始配置」，会分 PC、手机端、终端 指导进行配置。

## 一、 PC 端：Clash Verge (Rev) / Meta 配置

PC端利用Clash的策略组功能实现自动化分流。

1. **导入订阅**：从 EqualVPN 获取 Clash 订阅链接并导入。
2. **策略组设置**：
   - 找到 OpenAI / ChatGPT 策略组，手动指定为 **"专属纯净静态住宅节点"**。
   - 其他娱乐流量（如 YouTube）建议指定为 **"赠送的专线节点"** 以节省住宅流量。
3. **系统代理**：开启系统代理，建议使用 Rule (规则) 模式。

## 二、 手机端：Shadowrocket (小火箭) 配置 (核心)

采用"精细化白名单"逻辑。

### 1. 模式选择

- **首页** -> **全局路由** -> 选择 **配置 (Config)**。
- *注意：千万不要选"全局代理"，否则会极速消耗住宅流量且导致国内App卡顿。*

### 2. 建立代理白名单 (添加规则)

进入 配置 -> 点击当前配置文件旁的 i 图标 -> 规则 -> 添加规则：

| 类型 | 值 (Value) | 策略 (Policy) | 说明 |
| --- | --- | --- | --- |
| DOMAIN-KEYWORD | openai | EqualVPN-住宅节点 | 强制住宅IP，防降智 |
| DOMAIN-KEYWORD | claude | PROXY | 走默认代理 |
| DOMAIN-KEYWORD | discord | PROXY | 走默认代理 |
| DOMAIN-KEYWORD | twitter | PROXY | 包括 x.com |
| DOMAIN-KEYWORD | google | PROXY | 搜索业务 |
| DOMAIN-KEYWORD | ping0.cc | PROXY | 环境检测专用 |

### 3. 设置兜底规则 (最重要的一步)

- 找到规则列表最底部的 **FINAL**。
- 将其策略设置为 **DIRECT (直连)**。
- **效果**：除了上面白名单里的 App，其他所有流量（微信、美团、淘宝等）全部不走代理。

## 三、 实战避坑指南

### 1. 订阅格式不符
- **坑**：直接将 Clash 链接填入圈X（Quantumult X）会提示识别失败。
- **解**：圈X不支持 YAML，需使用订阅转换器或手动配置 VLESS 节点字符串。

### 2. 节点切换无效
- **坑**：在首页切换了节点，但日志显示依然走住宅IP。
- **解**：规则优先级高于手动选点。将 FINAL 设为 PROXY（跟随手动）或 DIRECT（白名单模式）即可解决。

### 3. 推特图片加载失败
- **坑**：能上推特但没图。
- **解**：推特图片域名是 twimg.com。建议单独检查规则包是否包含图片服务器域名。

## 四、 环境验证

配置完成后，务必访问 [ping0.cc](https://ping0.cc) 检查：

1. **IP地址**：应显示为 EqualVPN 的住宅 IP。
2. **ASN/类型**：应显示为 ISP 或 Residential。
3. **原生性检测**：各项 AI 服务可用性应为绿色。
